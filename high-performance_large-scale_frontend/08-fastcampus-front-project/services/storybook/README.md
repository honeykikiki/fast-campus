# storybook

# Dependencies 설명 및 설치 명령어

### @chromatic-com/storybook (^1.5.0)
- 설명: Chromatic과 통합된 Storybook 패키지로, UI 컴포넌트를 자동으로 테스트하고 시각적 회귀 테스트를 수행합니다.
- 설치 명령어:
  > yarn add @chromatic-com/storybook -D

### @fastcampus/react-components-layout (workspace:^)
- 설명: React 컴포넌트의 레이아웃을 구성하는 패키지입니다. workspace의 다른 패키지를 참조합니다.
- 설치 명령어:
  > yarn add @fastcampus/react-components-layout -D

### @fastcampus/themes (workspace:^)
- 설명: 프로젝트에서 사용되는 테마 및 스타일 설정을 모듈화한 패키지입니다. workspace의 다른 패키지를 참조합니다.
- 설치 명령어:
  > yarn add @fastcampus/themes -D

### @storybook/addon-essentials (^8.1.9)
- 설명: Storybook의 필수 애드온을 포함한 패키지로, Docs, Controls, Actions 등 다양한 기능을 제공합니다.
- 설치 명령어:
  > yarn add @storybook/addon-essentials -D

### @storybook/addon-interactions (^8.1.9)
- 설명: 상호작용 테스트를 위한 애드온으로, 사용자 인터랙션을 시뮬레이션하고 테스트할 수 있습니다.
- 설치 명령어:
  > yarn add @storybook/addon-interactions -D

### @storybook/addon-links (^8.1.9)
- 설명: Storybook 내에서 컴포넌트 간의 링크를 생성할 수 있는 애드온입니다.
- 설치 명령어:
  > yarn add @storybook/addon-links -D

### @storybook/addon-onboarding (^8.1.9)
- 설명: Storybook의 온보딩 애드온으로, 새로운 사용자들이 Storybook을 이해하고 사용하는 데 도움을 줍니다.
- 설치 명령어:
  > yarn add @storybook/addon-onboarding -D

### @storybook/blocks (^8.1.9)
- 설명: Storybook을 사용하여 UI를 구성할 때 필요한 기본 블록을 제공하는 패키지입니다.
- 설치 명령어:
  > yarn add @storybook/blocks -D

### @storybook/react (^8.1.9)
- 설명: React를 위한 Storybook 패키지로, React 컴포넌트를 Storybook 환경에서 개발하고 테스트할 수 있습니다.
- 설치 명령어:
  > yarn add @storybook/react -D

### @storybook/react-vite (^8.1.9)
- 설명: Vite를 사용하여 Storybook을 설정하고 번들링하는 패키지입니다.
- 설치 명령어:
  > yarn add @storybook/react-vite -D

### @storybook/test (^8.1.9)
- 설명: Storybook 내에서 테스트를 작성하고 실행할 수 있는 패키지입니다.
- 설치 명령어:
  > yarn add @storybook/test -D

### prop-types (^15.8.1)
- 설명: React 컴포넌트의 PropTypes를 정의하여 타입 검사를 수행할 수 있는 라이브러리입니다.
- 설치 명령어:
  > yarn add prop-types -D

### storybook (^8.1.9)
- 설명: Storybook의 핵심 패키지로, UI 컴포넌트를 독립적으로 개발하고 테스트할 수 있는 환경을 제공합니다.
- 설치 명령어:
  > yarn add storybook -D

### storybook-addon-root-attributes (^2.1.3)
- 설명: Storybook에서 루트 요소에 속성을 추가할 수 있는 애드온입니다.
- 설치 명령어:
  > yarn add storybook-addon-root-attributes -D

### vite (4.4.3)
- 설명: 빠르고 효율적인 프론트엔드 빌드 도구로, 개발 서버와 번들러 기능을 제공합니다.
- 설치 명령어:
  > yarn add vite -D
